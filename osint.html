OSINT Tools

OSINT tools are used for gathering publicly available data from a wide range of online sources. These tools help in investigations, social media analysis, and uncovering sensitive information that may be inadvertently exposed.

Linux

Spiderfoot

SpiderFoot is an OSINT automation tool that collects and analyzes data from over 200 sources, including social media, DNS records, and vulnerability databases. It supports API integrations for additional power.

The Harvester

This tool is specifically designed to gather information such as email addresses, subdomains, and IP addresses using sources like search engines and Shodan. It is widely used in the reconnaissance phase of penetration testing.

Censys

Censys is a powerful search engine that indexes publicly available data related to hosts and services on the internet. It allows users to search for exposed devices, domains, and servers based on specific criteria. Itâ€™s especially valuable for security researchers and penetration testers.

Windows

Maltego

Maltego is a sophisticated OSINT and link analysis tool. It helps visualize relationships between people, organizations, domains, and networks by leveraging various data sources. This tool is commonly used by cybersecurity professionals for threat analysis, investigations, and social network analysis.

Recon-ng

Recon-ng is a modular framework designed for reconnaissance. It provides an interface for automated collection of OSINT. Modules enable tasks such as gathering domain information, geolocation data, and profiling email addresses.

FOCA

FOCA (Fingerprinting Organizations with Collected Archives) is an OSINT tool primarily used for metadata extraction. It scans documents (PDFs, Word, Excel) for hidden metadata that might reveal sensitive information about the creators and systems. It's commonly used in security assessments to gather organizational details.

macOS

OSINT Framework

While browser-based, this tool is a curated collection of OSINT tools and techniques accessible via a simple interface. It guides users to specialized resources for investigations.

Sherlock

Sherlock searches for usernames across numerous social networks and websites, identifying accounts associated with a given username. This is particularly useful for investigations involving social media activity.

Scylla

Scylla is a comprehensive OSINT tool designed to search for online data about people and organizations. It searches through multiple data sources, including social networks, public records, and various other databases. It is user-friendly and effective for gathering detailed profiles.
